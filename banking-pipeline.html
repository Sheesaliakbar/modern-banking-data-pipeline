Layer 06 · Transform
dbt — Medallion Architecture
Bronze → Silver → Gold inside Snowflake
🥉
BRONZE
Raw CDC events — as loaded. Full history preserved.
🥈
SILVER
Dedup · Type casting · CDC op filtering · Nulls
🥇
GOLD
Balances · Fraud signals · Customer summaries
  • ✦ Silver handles CDC op filtering — INSERT, UPDATE, DELETE treated correctly
  • ✦ dbt tests enforce not_null, unique, referential integrity at every layer
  • ✦ Pipeline stops automatically if any test fails — bad data never reaches Gold
